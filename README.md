A Full Stack App

## Built With

### Front End

- Next.JS
- React.JS
- Styled-Component
- Material-UI
- React-cookie
- React Hook Form



### Back End

- Express
- MongoDB (Mongoose)
- JWT
- Redis

## How to Start

1. ```cd server```
2. ```npm install && npm run dev```
2. `cd ..`
3. ```cd client```
4. ```yarn install && yarn dev```
